Job Portal System

Full-Stack Application: Python Pyramid Backend + React Frontend

Project Structure

Backend: Python Pyramid + PostgreSQL API server at http://localhost:6543

Frontend: React.js + Vite client at http://localhost:3000

Database: PostgreSQL for persistent storage

Quick Start

Backend (Terminal 1):

cd backend && venv\Scripts\activate && python main.py

Frontend (Terminal 2):

cd frontend && npm install && npm run dev
